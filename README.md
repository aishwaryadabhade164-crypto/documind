# DocuMind – A Blog Writing Agent

DocuMind is a simple AI tool that creates full blog articles from a topic.  
It understands the topic, creates an outline, and writes a complete blog automatically.

## How to Use
1. Install requirements: `pip install -r requirements.txt`
2. Run: `python documind.py`

## Files
- documind.py → main code
- requirements.txt → dependencies
